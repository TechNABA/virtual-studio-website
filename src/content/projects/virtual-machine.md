- A virtual machine was created to host both the Perforce repositories and the Multi-User server.
- There is the possibility of expanding both the available bandwidth and the capacity of the virtual disks for the repositories.
- The virtual machine is connected to a special network separated from the campus network to avoid conflicts with other activities.
