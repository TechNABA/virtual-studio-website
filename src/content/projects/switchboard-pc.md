- The switchboard is used to control all compatible devices inside the studio.
- It serves as a media server for booting nDisplay instances and performance monitoring but also for booting Unreal instances on Remote PCs.
- We have integrated the Perforce repository system in order to synchronize on all render nodes the various projects.
- When it comes to performance capture we use the switchboard to trigger recording on all connected devices.
