- With the need to remotely control lights, we have implemented the DMX protocol. The lights are connected via DMX to an ETC console for professional control of fixtures and the creation of dedicated presets.
- We also implemented an ethernet connection within Unreal Engine to create a virtual copy of the control console and also get the actual data of the lights on the set.
