With Optitrack's optical tracking system and proprietary software, Motive, we are able to track multiple bodies simultaneously and stream the data in real-time to software such as Unreal Engine. This connection opens the door to a host of different possibilities including:

- Real-time retargeting of characters
- Live events with visual effects
- Previz for animated movies
- Motion analysis for scientific purposes
