- Implementation of Unreal Engine's Virtual Scouting system to pre-visualise scenes through headsets and make real-time changes.
- We have implemented the use of Scouting within Multi-User sessions so that everyone can collaborate with users from PCs.
- There is the possibility of using various headset models within the studio (HTC Vive, Oculus Quest, Valve Index) in order to be more comfortable during previsualization.
